## Скачать

Мы рекомендуем скачивать TG Wrist из Google Play:

[Скачать в Google Play](https://play.google.com/store/apps/details?id=com.tgwrist.app)

### Установка через ADB

> Не рекомендуется для большинства пользователей.  
> По возможности устанавливайте приложение из Google Play.

Если вы все же хотите установить APK вручную:

1. Скачайте APK из GitHub Releases.
2. Установите его через ADB:

```shell
adb install TGWrist.apk
```

## Возможности

- Отправка текстовых сообщений прямо с часов
- Получение текста, изображений, видео и других сообщений
- Самостоятельная работа без зависимости от телефона
- Своевременные уведомления о сообщениях через FCM
- Поддержка списка чатов, истории сообщений и предпросмотра медиа

## Детали дизайна

- Разработано специально для часов Wear OS
- Тщательно оптимизировано для небольших круглых экранов
- Используются макеты, адаптированные для устройств Wear OS
- Интерфейс в стиле Material Design 3 / Wear OS
- Плавная прокрутка и отзывчивое взаимодействие

## Конфиденциальность и прозрачность

- Защищает чувствительные локальные данные аккаунта с помощью Android Keystore / TEE
- Хранит данные аккаунта в локальном приватном хранилище приложения
- Удаляет локальные данные аккаунта после выхода
- Позволяет пользователям управлять включением сервисов Google, таких как FCM
- Исходный код опубликован для прозрачности и публичной проверки

## Лицензия

Исходный код этого проекта доступен для просмотра, но это не программное обеспечение с открытым исходным кодом.

Исходный код опубликован только для прозрачности, проверки безопасности и
публичного ознакомления. Без явного письменного разрешения нельзя копировать,
повторно использовать, изменять, распространять этот код или создавать на его основе
производные работы.
